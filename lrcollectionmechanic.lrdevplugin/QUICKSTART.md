# Quick Start Guide - Collection Mechanic

Get started with Collection Mechanic in 5 minutes!

## Installation (2 minutes)

### macOS
1. Copy the plugin folder to:
   ```
   ~/Library/Application Support/Adobe/Lightroom/Plugins/
   ```
2. Restart Lightroom Classic

### Windows
1. Copy the plugin folder to:
   ```
   C:\Users\[YourUsername]\AppData\Roaming\Adobe\Lightroom\Plugins\
   ```
2. Restart Lightroom Classic

## Your First Collection Set (1 minute)

1. In Lightroom, go to **Library** module
2. Right-click in the Collections panel
3. Select **Create Collection Set**
4. Name it (e.g., "My Project")
5. Click **Create**

## Create Your First Collections (2 minutes)

1. In Lightroom, go to **Library** → **Create Collections in Batch**
2. A dialog appears:
   - **Select Collection Set**: Choose "My Project"
   - **Enter Names**: Type:
     ```
     Before Photos
     After Photos
     Final Selects
     ```
3. Click **Dry Run** to preview
4. Click **Execute** to create
5. You'll see: "Successfully created 3 collection(s)"
6. Look in Collections panel - your 3 new collections are there!

## What's Next?

- **Try More Names**: Use special characters like slashes and colons - the plugin will clean them up
- **Create Larger Sets**: Create 20-50 collections at once - try it!
- **Read Examples**: Check [EXAMPLES.md](EXAMPLES.md) for real-world scenarios
- **Learn More**: See [README.md](README.md) for full documentation

## Keyboard Tips

- **Dry Run**: Preview changes before committing
- **Execute**: Create all collections instantly
- **Close**: Exit without making changes

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Plugin doesn't appear in menu | Restart Lightroom |
| "No Collection Sets Found" | Create a collection set first in Lightroom |
| Collections not showing up | Check if you selected the correct parent set |
